# TikTok 策略（8K）

## 內容類型

| 類型 | 範例 | 時長 |
|------|------|------|
| 快訊解讀 | 「比特幣剛剛爆倉 1 億美金，發生什麼事？」 | 30-60 秒 |
| 指標科普 | 「資金費率 0.1% 代表什麼？」 | 45-90 秒 |
| 迷因向 | 「空軍的一天」配上清算數據 | 15-30 秒 |
| 事件回顧 | 「2022 Luna 崩盤那天，市場是什麼樣子」 | 60-90 秒 |

## 發文策略

- **頻率**：每週 3-5 支
- **時間**：晚上 8-10 點（台灣高活躍時段）
- **字幕**：一定要有（很多人靜音看）

# 內容模板：10 種可複製格式

## 1. 每日 FGI 圖卡

```
標題：今日恐懼貪婪指數
數值：42（恐懼）
視覺：儀表盤 + 顏色
CTA：「加 LINE 收每日推播」
```

## 2. 清算快訊

```
標題：⚠️ 過去 1 小時 $1.2 億清算
數據：多/空比例、主要幣種
解讀：一句話結論
CTA：「即時警報 👉 @加密台灣」
```

## 3. 歷史對比

```
標題：今天的 FGI 15，上次這麼低是？
對比：2022 年 6 月 Luna 後
結果：30 天後 +23%
CTA：「更多歷史案例 👉 連結」
```

## 4. 指標教學 Carousel（5-7 頁）

```
P1: 封面「什麼是資金費率？」
P2: 簡單定義
P3: 高/低分別代表g什麼
P4: 歷史案例
P5: 如何在 CryptoTW 查看
P6: CTA
```

## 5. 迷因模板

```
市場情況 + 經典迷因圖
例：「FGI 10 的時候 vs FGI 90 的時候」
```

## 6. TikTok 快訊腳本

```
0-3 秒：Hook「比特幣剛剛發生大事」
3-15 秒：發生什麼
15-25 秒：這代表什麼
25-30 秒：CTA「追蹤看更多」
```

## 7. 週報 Carousel

```
P1: 封面「本週市場總結」
P2: 價格表現
P3: 關鍵指標變化
P4: 重大事件
P5: 下週觀察重點
P6: CTA
```

## 8. 互動貼文（Story）

```
投票：「你覺得下週 BTC 會？」
滑桿：「你現在有多恐懼？」
問答：「有什麼想問的？」
```

## 9. 產品功能介紹

```
螢幕錄影 + 字幕
展示：開啟 CryptoTW → 查指標 → 設警報
時長：30-60 秒
```

## 10. KOL/事件蹭流量

```
當天熱門事件 + 我們的數據視角
例：「川普說要買比特幣，FGI 從 X 跳到 Y」
```
