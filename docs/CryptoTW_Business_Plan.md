# 一、產品定位

## 1.1 核心定位
>
> **「讓散戶看見大戶看的數據」**

### 判斷層定位（Sub-Positioning）
>
> **「把機構用來判斷風險的位置，壓縮成散戶 3 秒能懂的結論。」**

這說明了：

- 為什麼不是 Coinglass → 我們給結論，不是給數據海
- 為什麼不是 ChatGPT → 我們有市場結構，不是通用模型

### 系統入口
>
> **CryptoTW Pro 是一個以 LINE 為核心入口的加密市場決策系統。**

---

## 1.2 核心價值主張

| 傳統工具 | CryptoTW Pro |
|---------|--------------|
| 數據分散、需切換多平台 | **一站式整合** 關鍵指標 |
| 英文介面、門檻高 | **全中文在地化**體驗 |
| 只看數值、不知道意義 | **語意化解讀** + 歷史案例 |
| 被動查詢 | **主動推播**異常警報 |
| **看完還是不知道該進不進** | **直接給你「偏多 / 偏空 / 觀望」結構** |

### 獨家優勢
>
> **「CryptoTW Pro 不只告訴你現在發生什麼，而是告訴你：過去發生這種事，市場通常怎麼走。」**
>
> **不是資料更多，而是更少但更關鍵。**

# 二、功能清單

## 2.0 功能分層設計

> [!IMPORTANT]
> CryptoTW Pro 的指標不是堆疊，而是有方法論的分層結構：

### 第一層：市場體感

**「今天市場在緊張還是放鬆」**

- 恐懼貪婪指數 (FGI)
- 資金費率
- 清算數據

### 第二層：結構判斷

**「是短線擁擠還是趨勢改變」**

- 未平倉合約 (OI)
- 期貨基差
- 多空比

### 第三層：資金方向

**「錢從哪來、要去哪」**

- ETF 資金流
- Coinbase 溢價
- 穩定幣市值

## 2.1 核心功能模組

### 📊 市場指標儀表板

| 指標 | 數據來源 | 決策意義 |
|------|---------|----------|
| 恐懼貪婪指數 | Coinglass | 市場情緒溫度計 |
| 資金費率 | Coinglass | 多空擁擠程度 |
| 未平倉合約 | Coinglass | 槓桿熱度警示 |
| 多空比 | Coinglass | 散戶反向指標 |
| 清算數據 | Coinglass | 極端波動信號 |
| ETF 資金流 | Coinglass | 機構動向 |
| Coinbase 溢價 | Coinglass | 美國需求強度 |
| 期貨基差 | Coinglass | 市場結構健康度 |
| 穩定幣市值 | Coinglass | 場外乾火藥 |

### 📈 指標深度頁面

- **當前位置**：即時數值 + 區間標籤（偏多/偏空/觀望）
- **4 區間視覺化**：彩色背景顯示高/低位
- **比特幣走勢對照**：指標與價格疊加
- **歷史案例連結**：過往類似情況的價格反應

### 📚 歷史事件回顧（王牌功能）
>
> **不是新聞回顧，而是價格反應回顧**

- 重大市場事件時間軸
- 7/30/90 天後漲跌幅
- AI 摘要與影響分析
- 連結相關指標當時數值

### 🔔 智能警報系統

- 指標異常自動推播
- LINE Bot 即時通知
- 只在「值得注意」時打擾你

### 🤖 LINE Bot（核心入口）

- 查詢指令：`/btc` `/fgi` `/funding`
- 主動推播市場異動
- 帳號綁定與 VIP 驗證
- **台灣市場的護城河**

# 四、商業模式

## 4.1 收費結構

| 方案 | 價格 | 對象 |
|------|------|------|
| **免費版** | $0 | 基礎指標、每日摘要 |
| **Pro 月費** | NT$299/月 | 全部指標、即時警報 |
| **Pro 年費** | NT$2,499/年 | 同上 + 2 個月免費 |
| **VIP 合作** | 免費 | 交易所合作用戶 |

### Pro 的價值定位
>
> **Pro 是為「不想被情緒洗掉的人」設計的。**

付費的三個理由（不是功能）：

1. **避開極端擁擠時刻**
2. **只在「值得注意」時被打擾**
3. **用歷史而不是情緒做決定**

## 4.2 收入來源

| 來源 | 佔比 |
|------|------|
| 訂閱收入 | 40% |
| 交易所聯盟導流 | 35% |
| 廣告/贊助 | 15% |
| 數據 API 授權 | 10% |

## 4.3 成本結構
