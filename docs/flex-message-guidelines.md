# CryptoTW Flex Message 設計規範

**原則**：簡潔、快速理解、不花俏

---

## 色彩系統

| 用途 | 色碼 |
|------|------|
| 主按鈕 | `#000000` |
| 漲幅/成功 | `#00B900` |
| 跌幅/錯誤 | `#D00000` |
| 警告/溢價 | `#ff8800` |
| 次要文字 | `#888888` |
| Binance | `#F0B90B` |

---

## 尺寸標準

- Bubble 尺寸：`kilo`
- 標題：`lg` + `bold`
- 內文：`sm`
- 說明文字：`xs` / `xxs`

---

## 範本類別

| 指令 | 功能 |
|------|------|
| `#BTC` | 價格卡片 |
| `#HOT` | 漲跌排行 |
| `#TWD` | 匯率快訊 |
| `#TWD 1000` | 換算結果 |

---

## 禁止事項

- ❌ 超過 2 個按鈕
- ❌ 長篇說明文字
- ❌ 未定義色彩
- ❌ 硬編碼 LIFF URL
