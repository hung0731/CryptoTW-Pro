# CryptoTW Flex Message 設計規範
## 加密台灣 Pro 品牌指南

---

## 色彩系統

| 用途 | 色碼 |
|------|------|
| **主色 (標題、主按鈕)** | `#1F1AD9` |
| **副按鈕** | `#000000` |
| 漲幅/成功 | `#00B900` |
| 跌幅/錯誤 | `#D00000` |
| 警告/溢價 | `#ff8800` |
| 次要文字 | `#888888` |

---

## 結構規範

每個 Flex Message 必須包含：
1. **品牌標籤**：`加密台灣 Pro` (xxs, #888888)
2. **標題**：主色 #1F1AD9, lg, bold
3. **按鈕**：第一 #1F1AD9，第二 #000000

---

## 範本類別

| 指令 | 功能 |
|------|------|
| `#BTC` | 價格卡片 |
| `#HOT` | 漲跌排行 |
| `#TWD` | 匯率快訊 |
| `#TWD 1000` | 換算結果 |
